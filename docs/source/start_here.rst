Start Here
==========

If you are opening the documentation for the first time, start with these repo-level guides:

- https://github.com/skygazer42/pyimgano/blob/main/docs/START_HERE.md
- https://github.com/skygazer42/pyimgano/blob/main/docs/STARTER_PATHS.md
- https://github.com/skygazer42/pyimgano/blob/main/docs/BENCHMARK_GETTING_STARTED.md
- https://github.com/skygazer42/pyimgano/blob/main/docs/CLI_REFERENCE.md

Recommended order:

1. Run ``pyimgano-doctor`` to check the environment and extras.
2. Run ``pyimgano-demo --smoke`` to exercise the shortest offline-safe workflow.
3. Use starter benchmark discovery if you want your first curated benchmark run.
