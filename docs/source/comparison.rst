Comparison (PyImgAno vs PyOD)
=============================

PyImgAno is designed for **visual anomaly detection** and industrial inspection workflows.
PyOD is a general-purpose outlier detection toolbox for tabular/feature data.

For a more detailed comparison, see:

- https://github.com/skygazer42/pyimgano/blob/main/docs/COMPARISON.md

High-level differences
----------------------

- PyImgAno emphasizes **image pipelines**, anomaly maps, and deploy-style JSONL inference.
- PyOD emphasizes a broad set of classical detectors and sklearn-like ergonomics.
