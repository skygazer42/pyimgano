Benchmarks
==========

PyImgAno provides two related layers:

- ``pyimgano-benchmark``: benchmark runs with standardized artifacts (JSON + per-image JSONL).
- ``pyimgano-train`` (workbench): recipe-driven runs focused on industrial workflows and portability.
