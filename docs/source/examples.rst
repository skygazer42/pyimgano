Examples
========

The repository contains runnable scripts under ``examples/``.

Browse:

- https://github.com/skygazer42/pyimgano/tree/main/examples

Highlights
----------

- ``examples/quick_start.py``: minimal detector usage.
- ``examples/industrial_tiling_infer.py``: high-resolution tiling inference.
- ``examples/industrial_infer_numpy.py``: numpy-first inference (decoded frames).
- ``examples/configs/*.json``: workbench config templates.

For CLI examples, see:

- https://github.com/skygazer42/pyimgano/blob/main/docs/CLI_REFERENCE.md
